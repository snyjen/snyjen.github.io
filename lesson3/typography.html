Typography

Heading Font: Playfair Display

I choose Playfair Display as the heading font because I like the more simple serif style, I think it goes along well with the name of the site.

Body Font: Open Sans

I choose Open Sans for my body font because it is an easy font to read and it is also a sans serif which is a nice contrast to the heading font which is a serif font.
